# Framework PYAUT

Raw framework for automation, preparing dataset and extracting value from information

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

* python 3.8.8
* requirements.txt

### Installing

* pyenv for python 3.8.8 installation:
    # installing pyenv for windows - https://github.com/pyenv-win/pyenv-win
    - (Windows PowerShell) $ Invoke-WebRequest -UseBasicParsing -Uri "https://raw.githubusercontent.com/pyenv-win/pyenv-win/master/pyenv-win/install-pyenv-win.ps1" -OutFile "./install-pyenv-win.ps1"; &"./install-pyenv-win.ps1"
    - $ cd "complete\path\project"
    - pyenv install 3.8.8
    - pyevnv versions
    - pyenv global 3.8.8
    - pyenv local 3.8.8
* venv for requirements.txt installation:
    - $ cd projet\full\path
    - $ source\.venv\Scripts\activate
    - $ py -m pip install --upgrade "pip<24.1"
    - $ pip install --trusted-host=pypi.org --trusted-host=files.pythonhosted.org --trusted-host=pypi.python.org -r requirements.txt
    - ! if creating venv is needed:
        . $ py -m venv .venv
